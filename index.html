Trillek Computer Toolkit

Computer development tools for Trillek's computer v0.2.0

Get it

Download the tar.gz files for your operating system. doc.tgz contains documentation about both assemblers and vlink.

Tools included

This consists on a selection an agrupation of TR3200 computer development tools, that are :

  • WaveAsm : An simple and straightforward assembler for TR3200 and other cpus.
  • VASM : A powerful macro-assembler that can generate object files or raw binary files.
  • VLINK : Linker paired with VASM to generate raw binary files with the final code.
  • SmallerC Compiler core : SmallerC allows to compile C code and generate assembly files to feed VASM.

Build

Simply do : 1. make git_submodule to initiate git submodules 2. make to build it

To generate Windows binary files (cross-compile), is necesary to have installed mingw32 and launch make with the apropiated CC value. For example make CC=i686-w64-mingw32-gcc LD=i686-w64-mingw32-gcc BINEXT=.exe or make CC=i586-mingw32msvc-gcc LD=i586-mingw32msvc-gcc BINEXT=.exe

Also, you can generate documentation files of vasm and vlink with make doc

Install

sudo make install would install in your system :

  • vasm-tr3200
  • vlink
  • smlrc-tr3200
  • WaveAsm
